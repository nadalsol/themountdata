Data stores

Identificar y dominar las fuentes necesarias para la creación de valor partir de los datos es el primer paso del data analytics journey.

Por eso, tener la capacidad de acceder de manera ágil y consistente a las fuentes de datos utilizando distintas técnicas para extraer, transformar y cargar los datos es clave.

De misma manera no debemos olvidar el establecimiento de procesos relacionados con la calidad de datos.

Desarrollos de ETL, APIs, Scraping.

Algoritmos de Calidad de datos Limpieza / Normalización / Deduplicación / Consolidación de datos.

Herramientas habituales Pentaho-Kettel, Jsoup… etc.
